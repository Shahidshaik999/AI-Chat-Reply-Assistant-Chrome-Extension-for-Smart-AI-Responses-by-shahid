HOW TO LOAD:
------------
1. Open Chrome
2. Go to: chrome://extensions/
3. Enable "Developer mode" (top-right)
4. Click "Load unpacked"
5. Select: ai-chat-assistant/frontend-extension
6. Done!

TESTING:
--------
1. After loading, go to: https://web.whatsapp.com
2. Open Console (F12)
3. You should see:
   - "AI Chat Assistant initialized"
   - "Running on WhatsApp Web"
   - "WhatsApp chat container found"

4. Send yourself a test message
5. AI suggestions should appear!

BACKEND:
--------
Make sure backend is running:
- Terminal: cd ai-chat-assistant/backend-api
- Run: .\venv\Scripts\activate
- Run: python main.py
- Server at: http://localhost:8000

TROUBLESHOOTING:
----------------
If extension doesn't load:
- Check you selected "frontend-extension" folder (not parent)
- Look for red errors in chrome://extensions/
- Click "Errors" button to see details

If no suggestions appear:
- Check backend is running (http://localhost:8000/health)
- Open browser console (F12) for errors
- Verify API_URL in contentScript.js
